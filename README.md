# 如何为本 Repo 提供信息？
## 途径
1. 在 issue 区提出 issue 并且指明补充内容
2. Fork 本 Repo 并修改，并提出 Merge 请求
3. 若你希望保持匿名，请将你要补充的内容发邮件到 tsinghuadocs@protonmail.com
## 要求
内容补充鼓励实名但不强制实名，只要求在内容开头添加“本人对以下内容负责”即可.
真诚地欢迎您提供更多的信息。
